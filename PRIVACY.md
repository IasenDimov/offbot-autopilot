# Privacy Policy for OffBot

**Effective Date: March 2026**

OffBot is a personal automation tool designed to manage content on Pinterest.

### 1. Data Collection
OffBot does not collect, store, or share any personal data from its users or third parties. It operates solely through the Pinterest API to post curated content.

### 2. Data Usage
The application only uses the necessary API tokens to access the owner's Pinterest account for automated pinning. No data is sold or transferred to third parties.

### 3. Contact
For any questions regarding this privacy policy, please contact the repository owner via GitHub.
